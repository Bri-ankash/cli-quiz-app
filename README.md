---

**Features âœ…**

- **Category-based quizzes**: Easy, Medium, Hard
- **Random question selection**: Each session presents 3 questions from the chosen category
- **Timer per question**: Default 15 seconds to answer each question
- **Colored terminal output**: Questions, correct answers, and incorrect answers highlighted
- **High scores**: Track scores per category with player names
- **Menu-driven interface**: Easy to navigate in the terminal
